# Issac-My-personal-Chatbot

## Issac is my personal chatbot which is used to do all the tasks like :
    * Open System Apps
    * Open Websites on google
    * Send Emails
    * Mathematic Calculations 
    * Othe all Informations 
 
## Required Libaries :

      * pyttsx3
      * webbrowser
      * smtplib
      * random
      * SpeechRecogntion
      * wikipedia
      * datetime
      * wolframalpha
      * subprocess
